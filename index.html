Document Structure : Defines the document type and version of HTML (like HTML5). : The root element of an HTML document. All other elements are contained within this element. : Contains meta-information about the document, such as the title, character set, linked resources (like CSS and JavaScript), and other metadata. : Contains the content of the HTML document, including text, images, links, forms, and other elements. Metadata Elements : Sets the title of the document, which is displayed in the browser tab. <meta>: Provides metadata such as description, keywords, author, and viewport settings. <link>: Defines a relationship between the current document and an external resource (most commonly used to link stylesheets). <style>: Embeds internal CSS styles within the HTML document. <base>: Specifies the base URL for all relative URLs in a document. <script>: Embeds or references JavaScript code. Content Sectioning <header>: Defines a header section for a document or a section, usually containing introductory content or navigational links. <nav>: Defines a set of navigational links. <section>: Defines a section in a document, such as chapters, headers, footers, or other parts. <article>: Defines independent, self-contained content that could be distributed or reused. <aside>: Represents content that is tangentially related to the main content, such as sidebars or callout boxes. <footer>: Defines a footer for a document or a section, usually containing metadata or navigational links. <main>: Specifies the main content of a document. <address>: Provides contact information for the author or owner of the document. <h1>, <h2>, <h3>, <h4>, <h5>, <h6>: Define HTML headings, from the highest rank (<h1>) to the lowest (<h6>). Text Content <p>: Defines a paragraph. <a>: Defines a hyperlink to another document or resource. <span>: Used to group inline elements for styling purposes. <div>: Defines a division or section in an HTML document, often used to group block elements for styling or layout. <blockquote>: Specifies a section that is quoted from another source. <pre>: Defines preformatted text that preserves whitespace and line breaks. <br>: Inserts a line break. <hr>: Creates a thematic break (horizontal rule) in the content. <strong>: Indicates strong importance; usually bold. <em>: Emphasizes text; usually italic. <mark>: Highlights text for reference purposes. <small>: Indicates less important text, usually displayed smaller. <del>: Represents text that has been deleted. <ins>: Represents text that has been inserted. <sup>: Specifies superscript text. <sub>: Specifies subscript text. <code>: Represents a fragment of computer code. <kbd>: Represents user input, typically keyboard input. <samp>: Represents sample output from a computer program. <var>: Represents a variable in a mathematical expression or programming context. Inline Text Elements <b>: Makes text bold without conveying any extra importance. <i>: Makes text italic without conveying any extra emphasis. <u>: Underlines text. <abbr>: Represents an abbreviation; may provide a tooltip with a full description using the title attribute. <cite>: Represents the title of a work, such as a book, song, or article. <bdi>: Isolates a span of text that might be formatted in a different direction than other text outside it. <bdo>: Overrides the current text direction. <q>: Defines a short inline quotation. <time>: Represents a specific period in time. Media Content <img>: Embeds an image. <audio>: Embeds sound content, such as music or other audio streams. <video>: Embeds a video. <source>: Specifies multiple media resources for media elements like <audio> and <video>. <track>: Specifies text tracks (like subtitles) for media elements. <picture>: Contains multiple <source> elements and one <img> element to provide versions of an image for different devices or screen sizes. <figure>: Represents self-contained content, such as an image with an optional caption. <figcaption>: Represents a caption or legend for a <figure> element. Form Elements <form>: Defines an HTML form for user input. <input>: Defines an input control. <textarea>: Defines a multiline input control (text area). <button>: Defines a clickable button. <select>: Defines a drop-down list. <option>: Defines an option in a drop-down list. <optgroup>: Groups related options in a drop-down list. <label>: Defines a label for an input element. <fieldset>: Groups related elements in a form. <legend>: Defines a caption for a <fieldset>. <datalist>: Specifies a list of pre-defined options for input controls. <output>: Represents the result of a calculation or user action. <progress>: Represents the completion progress of a task. <meter>: Represents a scalar measurement within a known range (e.g., disk usage). Table Elements <table>: Defines a table. <caption>: Specifies a caption for a table. <tr>: Defines a row in a table. <th>: Defines a header cell in a table. <td>: Defines a cell in a table. <thead>: Groups header content in a table. <tbody>: Groups the body content in a table. <tfoot>: Groups the footer content in a table. <col>: Specifies column properties for each column within a <colgroup>. <colgroup>: Specifies a group of columns in a table for formatting. List Elements <ul>: Defines an unordered list. <ol>: Defines an ordered list. <li>: Defines a list item. <dl>: Defines a description list. <dt>: Defines a term/name in a description list. <dd>: Describes a term/name in a description list. Embedded Content <iframe>: Embeds another HTML page within the current page. <embed>: Embeds an external application, typically multimedia content. <object>: Represents an external resource, which can be treated as an image, an HTML sub-document, or an external application. <param>: Defines parameters for <object>. Interactive Elements <details>: Represents additional details the user can view or hide. <summary>: Defines a visible heading for a <details> element. <dialog>: Represents a dialog box or other interactive component. <menu>: Defines a list or menu of commands. <menuitem>: Defines a command or item that a user can invoke from a popup menu (deprecated in HTML5). Scripting <noscript>: Defines alternative content to be displayed if a script is not supported or disabled. Obsolete/Deprecated Elements Some elements have been deprecated or are no longer recommended for use, such as <acronym>, <applet>, <basefont>, <big>, <blink>, <center>, <font>, <marquee>, <tt>, <frameset>, <noframes>, <isindex>, etc. This list covers the most common HTML tags and their purposes.
